色彩

中性而又实用的百搭色彩，满足页面开发需求的各种场景。
内置配色
azure
适合用于一些背景比如管理界面
blue
比较适合一些鲜艳色系的页面
primary
朱蓝色,一般用于按钮
success
一般用于按钮提交类
warning
暖色系，一般用于提示性元素
orange
鲜艳的修饰色彩
danger
比较引人注意的警示颜色
purple
富贵色,一般用于鲜艳色导航
这些色彩广泛应用于背景布置，按钮，表格等场景，系统色彩可以通过API内的生成模块自定义配置，方便快捷
白搭灰色系列
whitesmoke
seashell
ivory
gray
darkgray
yahei
inverse
一般用于背景、边框等,灰色系代表极简，几乎可以与任何元素搭配，不易形成视觉疲劳，且永远不会过时。
