ThinkUI是吸取了国内外框架的优点而封装的一套前端框架，体积轻盈，拆分自由，组件模块化，用户无论是开发较大的项目还是小型项目，都能利用UI快速的进行开发，CSS样式表和样式打印模块为用户提供了更自由的样式生成方法。

UI以移动端为优先，适配所有尺寸的电子设备，根据不同尺寸自动响应，符合现阶段项目的开发需求。

快速开始
获得ThinkUI后，将assets文件夹直接复制到您的项目中
<link href="assets/ThinkPlus/font-awesome/css/font-awesome.min.css" rel="stylesheet" />
<link href="assets/ThinkUi/css/Tui-mini.css" rel="stylesheet" />
<script src="assets/ThinkPlus/jquery/jquery-2.1.4.js"></script>
<script src="assets/ThinkUi/js/Tui.js"></script>
UI依赖font-awesome，因此如果您需要图标，则必须引入font-awesome文件
UI特效Tui.js全部依赖jquery，因此必须引入jquery，当然如果网站是纯静态的，那就无需了。
解决IE89 Media Queries 问题
IE8和IE9并不支持媒体查询（Media Queries），但你可以使用下面的补丁完美兼容！
     
<script src="assets/ThinkPlus/iesuport/html5.min.js"></script>
<script src="assets/ThinkPlus/iesuport/respond.min.js"></script>
